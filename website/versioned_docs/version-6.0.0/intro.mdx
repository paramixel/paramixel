# Composable action trees for Java testing

Paramixel is a Java 17+ test framework for building tests as explicit, composable action trees. A tree can run sequential work, parallel work, lifecycle setup and teardown, fixture-scoped instance work, assertions, delays, repeats, and timeouts.

The implementation is centered on a small public API:

- `org.paramixel.api.Runner` executes actions or discovered factories.
- `org.paramixel.api.action.Builder` produces immutable `Action` trees.
- `org.paramixel.api.action.Action` is the sealed reusable execution unit.
- `org.paramixel.api.Paramixel` contains discovery annotations.
- `org.paramixel.api.Configuration`, `Listener`, `Result`, `Context`, `Status`, and `Descriptor` expose runtime configuration, callbacks, and execution state.

## Quick start

```java
package examples;

import org.paramixel.api.Paramixel;
import org.paramixel.api.action.Action;
import org.paramixel.api.action.Sequence;
import org.paramixel.api.action.Step;

public final class SmokeTest {
    @Paramixel.Factory
    public static Action smoke() {
        return Sequence.builder("smoke")
                .child(Step.of("open", ctx -> System.out.println("open")))
                .child(Step.of("assert", ctx -> System.out.println("assert")))
                .build();
    }
}
```

Run discovered factories with the Maven plugin goal `paramixel:test`, or run programmatically with `Runner.defaultRunner()`.
